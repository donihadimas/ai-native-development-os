# Product Vision: Personal Habit Tracker

## Problem

Solo users often lose track of small daily habits because existing tools are either too complex or too passive.

## Target Users

- Primary user: individual knowledge workers who want a lightweight daily habit tracker.
- Secondary users: indie builders testing a simple productivity app idea.

## Value Proposition

A tiny habit tracker helps users define a habit, mark daily completion, and review streak progress without setup friction.

## MVP Scope

- Create a habit.
- List active habits.
- Mark a habit complete for today.
- Show current streak.

## Out of Scope

- Social sharing.
- Payments.
- Native mobile app.
- Advanced analytics.

## Success Metrics

- User outcome: users can create and complete a habit in under one minute.
- Product usage: at least one habit completion is recorded in a simulated flow.
- Engineering signal: first feature can be split into small FE/BE tasks.
