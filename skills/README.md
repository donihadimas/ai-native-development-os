# Skills

Skills are reusable operating procedures for AI coding agents. Each skill is tool-agnostic where possible and follows the standard `SKILL.md` format from `prd.md`.

Generator skills include a Clarification Gate. Agents should ask focused questions before writing final artifacts when the available context is too vague for accurate output.

V1 skills cover context management, product discovery, PRD generation, architecture design, ADR generation, task breakdown, implementation planning, testing, and code review.

V2 adds focused extensions for API contract design and generic backend API development.

V2.x adds UI/UX design, database migration planning, security review, and release management skills.
