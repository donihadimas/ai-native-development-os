# Release: v0.5.0 - Ponytail Integration And Repo Link Guidance

## Scope

- Included: Ponytail optional integration support, integration repository link guidance, CLI tests, integration docs, website docs, AIOS routing updates, and npm package version metadata.
- Excluded: new package dependencies, vendoring upstream integration source files, automatic installer execution without explicit user approval, and breaking CLI command changes.

## User-Visible Changes

- Users can enable Ponytail with `aios integration add ponytail` or through guided integration setup.
- `aios integration list` now shows repository links for RTK, Caveman, and Ponytail.
- Guided integration setup and guided `aios integration add` show upstream repository links before optional installer approval.
- Generated or updated AIOS projects can receive local Ponytail rules through `.aios/integrations/ponytail.md`.

## Developer-Visible Changes

- `cli/package.json` and `cli/package-lock.json` are bumped to `0.5.0`.
- Integration metadata now includes shared repository links and descriptions for RTK, Caveman, and Ponytail.
- CLI tests cover Ponytail behavior and repository link guidance.
- `CHANGELOG.md` and this release note document the `0.5.0` release scope.
