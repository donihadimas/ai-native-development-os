# Release: v0.4.1 - Artifact Lifecycle Bundle Fix

## Scope

- Included: npm package asset sync for the latest AIOS artifact lifecycle, task/plan indexes, index-first task discovery, summary-first routing, and release guidance updates.
- Excluded: new CLI behavior, dependency changes, and unrelated website or product documentation changes.

## User-Visible Changes

- New generated projects include lightweight task and plan indexes in the project skeleton.
- Installed `.aios` workflow kits include the artifact lifecycle reference and stricter routing guidance.
- AI agents receive clearer instructions to avoid scanning completed task/plan archives unless release, audit, traceability, or completed-work review requires it.

## Developer-Visible Changes

- `cli/package.json` and `cli/package-lock.json` are bumped to `0.4.1`.
- `CHANGELOG.md` and `RELEASE.md` document the patch release.
- The npm dry-run tarball includes `assets/aios-kit/references/artifact-lifecycle.md`, `assets/project-skeleton/docs/tasks/index.md`, and `assets/project-skeleton/docs/plans/index.md`.

## Validation Evidence

- Tests: `npm test` passed with 77 tests, 0 failed.
- Review: release readiness check confirmed `0.4.0` is already published and tagged, while the published `0.4.0` tarball is missing the new lifecycle assets.
- Manual checks: `npm pack --dry-run` passed; `git diff --check` passed.
